How Does it Work?

matchboxes. Each has a board on the cover and beads that decide the moves. When a sequence of moves leads to a win, then place back the beads plus extra. When draw put beads back, no reward. When loss, take beads out. Normally starts with a specific number of beads in each box, but I start with one of each in each box and have it learn from there. The idea is to make it fail proof, and to learn faster.
