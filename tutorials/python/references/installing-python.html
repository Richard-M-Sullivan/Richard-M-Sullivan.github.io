Installing Python

Download From the Python Website

First go to the official python website. Then click the downloads tab, followed by clicking the grey button containing the suggested download. This download is an installer that will set up the things needed to run python programs on your system.

Run the Installer

Now that the installer has downloaded, run it and go through the prompts. Note that it is important to know where you have python installed if you want to change that later. The installer will install a few things: the python interpreter, which is called the python launcher, and IDLE, which is an basic text editor that makes writing and running python files easier. It is pretty basic, and when you get more advanced you are likely to not use it, but in the beginning IDLE is a great editor to use as a beginner.

Additional Tools

Picking a Text Editor

A text editor is a program that is used for writing plain text files. This differs from programs like Microsoft Word that saves files in a rich text format. This means that Word embeds additional information into the file to tell the program how you want the text to be rendered. While this is good for writing papers that are visually pleasing, this is not what you want to have happen when writing programs. Make sure that the editor you are using edits plain text. Another thing is that you want an editor that will highlight words for you, which makes reading code easier on the eyes. Some editors that I recommend you look at are VisualStudio Code, IDLE, and Sublime. IDLE will be the easiest to use, and well be the only editor that I will describe how to use for brevity's sake. I personally use the Vim text editor, but I do not recommend this because it has a steep learning curve.

Writing A Python Program

From IDLE

Open IDLE, which is installed where the python launcher is. You will see a window pop up called the shell. The shell is not used for editing python files. It is a window where the output of a running python program will display information. To create a new file, click file in the top left corner of your screen. You should see a new window pop up called untitled. Then click file, save as, and give it the name and location to save your program. Before finalizing the name of your file, make sure to look at the section below called Naming the File for advice on file naming conventions. Now you are ready to write some code!

From Another Text Editor

Create a file to hold your program. Make sure the file name aligns with the advice in the Naming the File section below. Now that your file has been created, open the file up in your text editor. You are now ready to write some code!

Naming the File

When you save your program make sure that you give it a meaningful name. Things like app and program don't offer much in helping you to know what is written in the file. Also make sure that you don't have any spaces in your file names. While this sounds silly, spaces are a pain when you start using the terminal, so I recommend either using camel case (thisIsAnExampleOfCamelCase) or using _ and - in place of a space (this_is_another_example). Also, when you decide what you want to do, be consistent so that you don't have to think about spelling when you are searching for a file name. Lastly, when you save your file, make sure that it has the extension .py. This tells the computer or any program opening the file that it is a python file, which is important for your text editor to highlight things correctly.

Running a Python File

From IDLE

Make sure to save your program, and then click run, and click run module. Your program should run and display its output to the shell. You can also press f5 to do this with a shortcut.

From Another Text Editor

If your text editor, like VS Code, has the ability to run the file from the editor, then do that. If not, you will run your file in the terminal. To do this, open the terminal, navigate to the location where your program is, and finally type in the command python3 nameOfYourFile.py. You should see the output of your program get written in the terminal. If you don't know terminal commands, you will have to look them up. I recommend looking at how to change directories, how to list the files in a directory, and how to run, create, and delete files. After learning those commands you should be able to do everything needed to complete the rest of the tutorials. The terminal is not as scary as it seems, and you might come to enjoy using it once you get familiar with some of its commands.

How Does Python Work

What is Programming

Programming is solving problems using computers. Kinda vague, I know, but the types of problems that programmers solve cover every topic imaginable! From a high level, a program is a set of instructions that tell a computer what to do. In the olden days people used to write in machine code, but this was complicated and error prone, so people wrote programs called compilers that allow people to write programs in human readable code, that gets translated into machine code. This is how languages like C work. You write a C program, and then you use a compiler to generate an executable, which is the machine code instructions that the computer can read. Now that computers are so fast, a program called an interpreter is used that reads a program and translates it into machine instructions on the fly. this is how python works.

The Python Interpreter

Python is an interpreted language. What this means is that to run a python program you need to have the python interpreter downloaded on the computer that you want to run the program on. Then the interpreter will read your python file line by line and tell the computer what to do. This makes python programs highly portable because they can be run on any computer that has a python interpreter - even if the underlying architectures of the computers are different.

Why do it This Way

This is benificial because it makes programs portable in that they will run on any computer that has the python interpreter installed. Very nice for sharing programs with collegues. Another benifit is that programs can be run immediately. Non-interpreted languages need to be compiled. This means that all the translation has to be done before you can run the program. For large programs this can take a long time, and you have to re-compile it every time that you make changes! With an interpreter there is no need to wait, and changes can be made in real time!

Common Questions

What can I do With Python?

A better question is, "what can't be done with python?". Python is one of the most widely used programming languages, and is used for all kinds of applications. It is used for scientific modeling, machine learning, web development, and even for making games. The only limit to what can be achieved with python is your imagination.

Isn't Python Considered Slow?

Yes, python as a language is very slow when compared with other languages like c/c++ or Java. That being said, computers are very fast these days, and it is ok that a language is slow. One way to think about it is that it is ok that you are not driving a race car because there are speed limits imposed on drivers. Having a faster car doesn't necessarily equal a good experience. One of the benifits of choosing python as a language is that it is easy to learn, which is why it widely adopted in scientific communities. Python is also very simple to write because the complexities of managing memory is done automatically, this makes writing programs way faster, which is why companies like it. Python allows a programmers to go from idea to product as fast as possible. Also, some libraries that you can use in your python code are written in faster languages such as c/c++, which makes python just as fast for certain tasks!

Do I Have to be a Genious to Become a Programmer?

Nope! All you need is a strong sense of curiosity, and to spend the time to practice. For the best results, practice every day! Your brain is a muscle, and over time you will find that the same problems come all the time. Being a good programmer is being able to identify those patters so you can build an intuition for how somthing is likely to be solved.
