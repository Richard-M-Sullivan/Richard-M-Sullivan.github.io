What is a Statement?

How Programs are Read in Python

Python reads code in the same way that one would read a story - from left to right and top to bottom. In English we split up ideas into sentences. These are units of text that express a complete thought. In Python, we use lines to split up our instructions. The Python interpreter will read our instructions one line at a time to do work.

Types of Statements

A statement is a line of code that does something. Examples of types of statements are as follows:

  1. Assign a value to a variable
  2. Change the value of a variable
  3. Check a condition, to run a block of code
  4. Call a function

This list expresses all the things that can be done in Python. In subsequent chapters you will learn the different ways of performing those tasks. In our hello world program, the instruction print("hello world") was a function call, and is a valid statement. If you wanted to print more items, you would write another print statement on the next line below.

Key Words

In Python there are words that have special meaning. These reserved words are not allowed to be used for any purpose other than what Python uses them for, other wise the interpreter would get confused. These key words are used to build up the types of statements described above and are broken into two categories: Structural indicators and operators.

Operators

Think of an operator as being like a math symbol. It is used to change the value of data and variables. Below are lists of the different types of operators and what they do.

Arithmetic Operators

These are symbols that take two values and combine them into a new value.

  • + : The addition operator adds two numbers
  • - : The difference operator subtracts the right number from the left
  • * : The multiplication operator multiplies two numbers together
  • ** : The Exponent operator raises the left number to the power of the right number
  • / : The division operator divides the left number by the right
  • // : The floor division operator divides the left number by the right, but discards the remainder
  • % : The modulus operator divides the left number by the right, but only keeps the remainder
  • () : The parenthesis operator evaluates what is inside of them before the outside

Comparison Operators

These are symbols that take two values and compares them. If the comparison is true, then it gives back the value True, and if not then it gives back the value False.
