Variables

Every program takes in data, makes changes to it, and gives back the results. This holds true for all programming applications and languages. In Python, variables are a way for the computer to remember the results of a calculation.

What is Data

Data is a bit of a vacuous term, but all it means is the information that is used inside of a program. In Python there are only a few kinds of data that we work with, and these are called data types.

Data Types

There are three main data types in Python: Integers, Floats, and Strings.

  • Integers: These are positive or negative whole numbers --> ..., -2, -1, 0, 1, 2, ...
  • Floats: These are all decimal numbers --> ..., -3.766, -1.56, 0.0, 1.2234, 109.3455, ...
  • Strings: these are sequences of characters --> 'hello', 'more text', 'even more text!, ...
